# correttore-ortografico
semplice  app per correggere ortografia e grammatica del testo in input
Correttore Ortografico AI

Una web app semplice ed elegante per correggere i tuoi testi usando l'intelligenza artificiale di Google Gemini.

🚀 Caricamento Manuale su GitHub

Per mettere online questa app gratuitamente:

Accedi a GitHub e crea un nuovo repository chiamato correttore-ortografico.

Assicurati che sia Public.

Clicca su "uploading an existing file".

Trascina il file index.html che trovi in questo progetto.

Fai il "Commit changes".

Vai nelle Settings del repository su GitHub.

Nel menu a sinistra, clicca su Pages.

Sotto "Build and deployment" > "Branch", seleziona main e clicca Save.

Dopo un minuto, il tuo sito sarà attivo all'indirizzo: https://tuo-username.github.io/correttore-ortografico/

🔑 Requisiti

Per funzionare, l'app richiede una Gemini API Key che puoi ottenere gratuitamente su Google AI Studio.
